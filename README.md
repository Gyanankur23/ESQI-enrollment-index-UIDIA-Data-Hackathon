# ESQI-enrollment-index-UIDIA-Data-Hackathon
Enrollment Stress &amp; Quality Index (ESQI) — Composite early-warning metric for Aadhaar enrollment systems, built at the UIDIA Data Hackathon. Combines stress, volume, and biometric activity (0–100 scale) to track resilience under load, with data sources, trends, anomalies, and recommendations.
